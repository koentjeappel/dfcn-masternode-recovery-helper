# DeFCoN Masternode Recovery Helper

Cautious recovery helper for DeFCoN masternodes with trusted addnode support.

## Planned features

- Trusted addnode list loaded from a separate file
- Optional block height checks for trusted nodes
- Careful daemon stop and restart handling
- Optional peer and cache cleanup
- Temporary trusted-peer mode
- Return to normal mode after recovery window
- Clear prompts and status messages

## Files

- `dfcn-mn-recovery.sh` - main recovery script
- `trusted_addnodes.txt` - trusted addnode list
